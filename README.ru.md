## Обзор

Flute — гибкая система управления контентом, созданная специально для игровых сообществ. Построена на современном PHP с модульной архитектурой и предоставляет всё необходимое для управления серверами CS2, CS:GO, Minecraft и других игр.

<br>

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/flute-cms/cms.git

# Установить зависимости
composer install

# Следовать веб-установщику
```

<br>

## Возможности

**Модульная архитектура** — расширение функционала плагинами и темами

**Интеграция с игровыми серверами** — нативная поддержка Source, GoldSrc и Minecraft

**Современный стек** — PHP 8.2+, компоненты Symfony, Cycle ORM

**Панель администратора** — полноценный интерфейс управления

**Платёжная система** — встроенная поддержка множества платёжных шлюзов

<br>

## Требования

- PHP 8.2+
- MySQL 5.7+ / MariaDB 10.3+ / PostgreSQL 12+
- Composer
