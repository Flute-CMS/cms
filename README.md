## Overview

Flute is a flexible content management system designed specifically for game server communities. Built with modern PHP and a modular architecture, it provides everything you need to manage CS2, CS:GO, Minecraft, and other game servers.

<br>

## Quick Start

```bash
# Clone the repository
git clone https://github.com/flute-cms/cms.git

# Install dependencies
composer install

# Follow the web installer
```

<br>

## Features

**Modular Architecture** — Extend functionality with plugins and themes

**Game Server Integration** — Native support for Source, GoldSrc, and Minecraft servers

**Modern Stack** — PHP 8.2+, Symfony components, Cycle ORM

**Admin Panel** — Comprehensive management interface

**Payment System** — Built-in support for multiple payment gateways

<br>

## Requirements

- PHP 8.2+
- MySQL 5.7+ / MariaDB 10.3+ / PostgreSQL 12+
- Composer
